# HTML, advanced

![Homepage wireframe](./wireframe.jpg)

## Description

This project is the first step in building a webpage from scratch based on a designer file (Figma). The focus here is purely on **HTML structure and semantics** — no CSS, no styling, just clean, valid, W3C-compliant markup.

The goal is to translate a wireframe and design specification into a well-structured HTML document, building a strong foundation that styling and interactivity will later be layered on top of.


## Requirements

- All files end with a new line
- A `README.md` file at the root of the project folder is mandatory
- No external libraries — built with **only HTML, CSS, and JavaScript** (no NodeJS, React, VueJS, Bootstrap, etc.)
